# Finance Snapshot (iOS App)

A modern personal finance tracking app built with SwiftUI that helps users monitor, analyze, and understand their spending behavior.

## 🚀 Features
- Add, edit, and delete expenses
- Custom categories for flexible tracking
- Monthly filtering (switch between months)
- Category-wise spending visualization (charts)
- Smart insights:
  - Top spending category
  - Highest expense
  - Average expense
- Weekly trend analysis with percentage change (↑ ↓)
- Budget tracking with real-time warning
- Smooth UI with animations
- Persistent data storage using SwiftData

## 🛠 Tech Stack
- SwiftUI
- SwiftData
- Charts Framework

## 📊 What makes this project different
This is not just a basic CRUD app.  
It includes data-driven insights and trend analysis to help users make better financial decisions.
